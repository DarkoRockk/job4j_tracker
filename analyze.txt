2. jconsole
На мой взгляд данная утилита является наиболее информативной, т.к. обладает графическим интерфейсом, что очень удобно.
Вкладка Overview - мы можем наблюдать сводную информацию графиках:
Heap Memory Usage
Threads
Classes
CPU Usage

Мы так же можем посмотреть более подробную информацию по каждой из частей, переключаясь по верхним вкладкам.
Memory - показывает заполнение пулов типа, а так же их очищение. Можно переключаться между пулами. В данный момент Eden у меня пустой, в то время как Old Generation активно растёт.
Threads - информация о работающих потоках и о их пиковом значении. В моем случае их было 16 (пик), сейчас стабильно 14, все потоки указаны в списке внизу.
Classes - график показывает кол-во загруженных классов в данный моммент. (Синяя линия) и кол-во всего загруженных (Красная линия).
VM Summary - показывает сводную информацию со всем значениями.
